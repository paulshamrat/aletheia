# Aletheia Technical Knowledge Base

Welcome to **Aletheia**, a collection of technical guides, research pipelines, and documentation.

## Overview
This wiki is built using [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) and serves as a central repository for:
- **Data Sync & Workflows**: Rclone setups and data management.
- **System Monitoring**: Linux performance and server metrics.
- **HPC & Environments**: Palmetto cluster guides and Conda environments.
- **Research Pipelines**: Molecular biology and structural annotation workflows.

## Features
- **Modern UI**: Dark/Light mode support.
- **Fast Search**: Built-in search functionality.
- **Clean Structure**: Organized navigation tabs and sections.
